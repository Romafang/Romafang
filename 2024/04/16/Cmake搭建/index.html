如何使用VS2022远程连接ubuntu使用Cmake

安装Cmake

  • Windows端

  • 首先在CMake官网下载安装包，官网地址：https://cmake.org/，点击Download。

  • 安装完毕，进行测试。
    打开cmd黑窗，输入命令查看版本

    cmake /V

  • Ubuntu

  • 方法1，使用apt（不建议，版本过低）
    sudo apt-get install cmake

  • 方法2，官网下载压缩包根据版本下载后缀是.tar.gz（可用wget下载）

    执行下面命令
    tar -xzvf cmake-3.xx.x-Linux-x86_64.tar.gz
    看自己版本

    如何再进入解压的文件夹里执行
    ./bootstrap
    或者
    ./configure

    再执行
    make -j8

    sudo make install

    最后检查版本是否安装成功

    cmake –version

使用Cmake远程连接Ubuntu要求两端都要有Cmake

建立连接及一些设置

然后在VS2022新建一个Cmake项目

在工具栏里点击工具-》选项》跨平台-》连接管理器添加连接

添加连接 HostName是Ubuntu的ip地址，username是Ubuntu的账号名，但是Cmake在远程连接执行的时候需要ninja的，需要管理员权限，最好使用管理员账号

然后在连接管理器下面的远程标头管理器中点击下载或者更新

再更改一些设置

选择Cmake-》常规

选择从不使用cmake预设

打开项目的cmake设置

点击红叉删除默认的再点击加号添加你所需的配置，例如我的Linux-GCC-Debug

选择启动项运行即可，可以自己更改CmakeList.txt根据自己所需
